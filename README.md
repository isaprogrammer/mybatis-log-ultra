## 核心功能

### 1. 关键词过滤 SQL
输入关键词（如表名、方法名、SQL 片段），实时筛选出你关心的 SQL 语句，告别在大量日志中大海捞针。

### 2. 一键跳转到源码
点击任意 SQL，直接跳转到执行该 SQL 的代码行，快速定位业务逻辑。

### 3. 多维度筛选
- **按请求筛选**：根据 HTTP 请求隔离 SQL，查看单次请求涉及的所有数据库操作
- **按关键词筛选**：快速定位特定 SQL
- **按函数筛选**：按 Mapper 方法名分组查看

## 快速上手

无需任何配置，安装后即可使用。
